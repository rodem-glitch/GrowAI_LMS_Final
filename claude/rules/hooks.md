# Hooks (훅) - 선택/개념 문서

이 문서는 `everything-claude-code/rules/hooks.md`의 “자동으로 실수를 막는 장치” 아이디어를 참고했습니다.

> 주의: 이 저장소의 문서만으로 “자동 훅”이 바로 실행되지는 않습니다.
> 다만 팀/환경에 따라 별도 훅 시스템을 붙일 때 참고할 수 있도록 개념만 정리합니다.

---

## 훅으로 막고 싶은 실수 예시
- 하드코딩 비밀값 커밋
- `project/` 수정 후 빌드 누락
- 의도치 않은 대규모 파일 변경/삭제
- 수정 파일에서 `console.log` 같은 디버그 코드 잔존
