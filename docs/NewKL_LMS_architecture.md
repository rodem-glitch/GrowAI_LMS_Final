## **1\. 개요**

한국폴리텍대학의 학사데이터 보안 및 운영 비용의 획기적 절감을 위해, 외부 API 의존도가 전혀 없는 **100% 온프레미스(On-premise) 기반의 GrowAI 시스템**을 구축하고자 함.

## **2\. 참조 모델 비교 분석 (Full Cloud vs Hybrid vs 100% On-premise)**

본 안건의 채택 근거를 위해 기존 검토된 모델들과의 비교 수치를 제시함.

| 비교 항목 | \[참조2\] Full Cloud | \[참조1\] Hybrid | \[본안\] 100% On-premise |
| :---- | :---- | :---- | :---- |
| **운영 방식** | GCP Gemini API 전담 | GCP \+ Legacy 서버 혼합 | **자체 GPU 서버 \+ 오픈소스 LLM** |
| **데이터 보안** | 외부 클라우드 전송 | 주요 데이터 내부 저장 | **망 분리 및 완전 폐쇄형 운영** |
| **월 고정 비용** | 약 $9,000 (고비용) | 약 $2,500 (중비용) | **약 20만 원 (전기료/유지비)** |
| **확장성** | 비용 비례적 증가 | 비용 완만한 증가 | **비용 증가 없는 무제한 확장** |

## **3\. 100% 온프레미스 모델의 핵심 특징 및 장점**

### **가. 독보적인 경제성 (Zero-Variable Cost)**

* **오픈소스 생태계 활용:** vLLM, Gemma 2, Milvus 등 검증된 오픈소스 기술을 내재화하여 **매달 발생하는 수천 달러의 API 과금을 전액 제거**함.  
* **TCO(총소유비용) 극소화:** 초기 셋팅 이후 추가 비용이 거의 발생하지 않아, 사용자 수가 늘어날수록 1인당 운영 단가가 0원에 수렴함.

### **나. 완벽한 데이터 주권 및 보안**

* **폐쇄망 운영:** 모든 학사 정보, 학생 상담 기록, 실습 영상이 학교 내부 망 내에서 처리되어 **정보 유출 가능성을 원천 차단**함.  
* **규제 준수:** 공공기관 및 교육기관의 엄격한 데이터 보안 가이드라인을 100% 충족함.

### **다. 무제한 커스터마이징 (Full-Control)**

* **전공 특화 튜닝:** 폴리텍대학 특유의 기술 용어 및 실습 매뉴얼을 제한 없이 학습(Fine-tuning)시켜 클라우드 대비 높은 정확도 구현.  
* **플랫폼 독립:** 특정 빅테크 기업의 정책 변화나 서비스 중단 리스크로부터 완벽히 자유로움.
