```
[메인] → [과정목록] → [과정상세] → [수강신청/결제]

[마이페이지] ← [수강완료] ← [학습실] ← [수강중 과정]
     ↓
[수료증 발급]
```

```
     ↓              ↓
[토론 참여]    [설문 응답]
     ↓
[Q&A 질문]
```
